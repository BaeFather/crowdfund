# 크라우드펀딩 플랫폼

---

## 프로젝트 개요

투자자와 차입자를 온라인으로 연결하는 **P2P 크라우드펀딩(온라인투자연계금융)** 플랫폼입니다.  
금융위원회 등록 온투업자로, 온라인투자연계금융업 및 이용자 보호에 관한 법률(온투법)에 따라 운영됩니다.

### 주요 기능

- **투자 상품 운영**: 부동산 담보 / 동산 담보 / 개인신용 / 사업자신용 / 포트폴리오 / 기타신용
- **투자자 회원 관리**: 일반 / 소득적격 / 전문 / 법인 투자자 유형별 투자 한도 관리
- **예치금 관리**: 신한은행 가상계좌 기반 예치금 입출금 처리
- **원리금 상환 처리**: 만기일시 / 원리금균등 / 원금균등 방식 자동 정산
- **자동투자**: 조건 설정 기반 자동투자 기능
- **KYC/AML**: 신분증 OCR(유스비), 실명확인, WLF(워치리스트 필터링)
- **P2P 중앙기록관리**: 금융위원회 P2PCTR 시스템 API 연동
- **모바일 앱 연동**: Android(Google Play) / iOS(App Store) 앱 지원
- **신디케이션**: 핀크, 올리고, 부동산114, 아이템베이 등 제휴사 연동

## 서버 인프라

```
[LB] 192.168.0.56
  ├─ www1  192.168.0.8
  └─ www2  192.168.0.91
```

- 서버 배포 경로: `/home/crowdfund/public_html/`
- `config.service.php` 변경은 manager 서버에서 수행 → crontab rsync로 www1/www2 자동 동기화
- 세션 경로: `data/session/` (도메인별 분리)
- 상품 목록 캐시: `data/cache/productList-{active|popular|latest}.php` (60초 TTL)

├── AML/                    # KYC/AML, WLF(워치리스트 필터링)

├── company/                # 회사소개, 이용약관, 개인정보처리방침
├── api/                    # 내부 REST API (회원 인증 등)
├── adm/                    # 관리자 페이지
├── syndicate/              # 제휴사 연동 (핀크, 올리고, 부동산114 등)
├── lib/                    # 공용 라이브러리
├── theme/2018/             # 활성 테마 (PC + 모바일)
└── bbs/                    # 그누보드 게시판

## 외부 연동 서비스

| 서비스 | 용도 |
|--------|------|
| **P2PCTR** (`openapi.p2pcenter.or.kr`) | 금융위원회 P2P 중앙기록관리 API |
| **신한은행 InsideBank** | 가상계좌 발급 / 대출 실행 / 원리금 지급 |
| **세틀뱅크** (`npay.settlebank.co.kr`) | 계좌 1원 인증 (계좌점유인증) |
| **유스비(USEB)** | 신분증 OCR 및 진위 확인 |
| **하이픈(Hyphen)** | 본인인증 |
| **나이스평가정보(NICE)** | 실명 확인 / 계좌 인증 |
| **INIpay50** | 온라인 결제 |
| **ext.hellofunding.co.kr** | 내부 확장 API 서버 |
